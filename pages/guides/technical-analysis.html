技术分析基础

技术分析是通过研究价格图表和交易量来预测未来价格走势的方法。它基于三个核心假设：

市场行为包含一切

所有影响价格的因素都已经反映在价格走势中，包括基本面、政治、心理等因素。

价格按趋势运动

价格运动不是随机的，而是按照一定的趋势发展，趋势一旦形成就会持续一段时间。

历史会重演

市场参与者的心理和行为模式是相似的，因此历史价格模式会重复出现。

K线图基础

K线的构成

最高价 收盘价 开盘价 最低价

K线要素

  • 实体：开盘价和收盘价之间的区域
  • 上影线：最高价到实体上端的线段
  • 下影线：最低价到实体下端的线段
  • 颜色：绿色(阳线)表示上涨，红色(阴线)表示下跌

常见K线形态

十字星

开盘价和收盘价几乎相等，表示买卖双方力量均衡，通常预示趋势可能反转。

重要性：★★★★☆

锤子线

实体较小，下影线很长，出现在下跌趋势末期，通常是看涨反转信号。

重要性：★★★★☆

流星线

实体较小，上影线很长，出现在上涨趋势末期，通常是看跌反转信号。

重要性：★★★★☆

吞没形态

第二根K线完全包含第一根K线，是强烈的反转信号。

重要性：★★★★★

时间周期分析

短期交易 (1分钟-1小时)

优点：
  • 快速获利机会
  • 高频交易可能
  • 资金周转快
缺点：
  • 噪音较多
  • 交易成本高
  • 心理压力大

中期交易 (4小时-日线)

优点：
  • 信号相对可靠
  • 适合上班族
  • 趋势较明确
缺点：
  • 获利周期较长
  • 需要更多耐心
  • 资金占用时间长

长期投资 (周线-月线)

优点：
  • 大趋势明确
  • 交易成本低
  • 心理压力小
缺点：
  • 资金占用时间长
  • 机会相对较少
  • 需要强大耐心

图表形态

反转形态

头肩顶/头肩底

最可靠的反转形态之一，由三个峰值组成，中间的峰值最高(头部)，两边的峰值较低且大致相等(肩部)。

交易规则：
  • 确认信号：价格跌破颈线
  • 目标价位：头部到颈线的距离
  • 止损设置：右肩上方
  • 成交量：突破时应放量确认

双顶/双底

由两个大致相等的峰值组成，是常见的反转形态。双顶出现在上涨趋势末期，双底出现在下跌趋势末期。

交易规则：
  • 确认信号：价格突破颈线
  • 目标价位：峰值到颈线的距离
  • 止损设置：第二个峰值附近
  • 时间间隔：两个峰值间隔不宜太短

持续形态

三角形整理

上升三角形

上边线水平，下边线上倾，通常向上突破

下降三角形

下边线水平，上边线下倾，通常向下突破

对称三角形

上下边线收敛，突破方向不确定

矩形整理

价格在水平的支撑和阻力之间震荡，形成矩形区域。突破后通常延续原趋势。

交易策略：
  • 区间内高抛低吸
  • 突破后跟随趋势
  • 假突破要及时止损

技术指标

趋势指标

移动平均线 (MA)

最基础也是最重要的趋势指标，通过平滑价格波动来识别趋势方向。

简单移动平均线 (SMA)

计算公式：(P1 + P2 + ... + Pn) / n

特点：对所有价格给予相等权重

指数移动平均线 (EMA)

对近期价格给予更高权重

特点：反应更敏感，滞后性较小

交易策略：
  • 金叉：短期MA上穿长期MA，买入信号
  • 死叉：短期MA下穿长期MA，卖出信号
  • 支撑阻力：MA可作为动态支撑阻力位
  • 趋势确认：价格在MA上方为上涨趋势

MACD指标

由快线(DIF)、慢线(DEA)和柱状图(MACD)组成，用于判断趋势变化。

  • DIF线：12日EMA - 26日EMA
  • DEA线：DIF的9日EMA
  • MACD柱：(DIF - DEA) × 2
交易信号：
  • 金叉：DIF上穿DEA，买入信号
  • 死叉：DIF下穿DEA，卖出信号
  • 零轴：MACD在零轴上方为多头市场
  • 背离：价格与MACD走势相反

震荡指标

相对强弱指数 (RSI)

衡量价格变动速度和幅度的震荡指标，取值范围0-100。

超买区 (RSI > 70)

价格可能面临回调压力

中性区 (30 < RSI < 70)

正常波动范围

超卖区 (RSI < 30)

价格可能出现反弹

交易策略：
  • RSI从超卖区向上突破30，考虑买入
  • RSI从超买区向下跌破70，考虑卖出
  • 寻找RSI与价格的背离信号
  • 结合其他指标确认信号

布林带 (Bollinger Bands)

由中轨(20日MA)、上轨(中轨+2倍标准差)、下轨(中轨-2倍标准差)组成。

  • 带宽：反映市场波动性
  • 位置：价格相对位置显示强弱
  • 收缩：预示大行情即将到来
  • 扩张：表示波动性增加
交易策略：
  • 价格触及上轨考虑卖出
  • 价格触及下轨考虑买入
  • 突破上轨后可能继续上涨
  • 跌破下轨后可能继续下跌

指标组合使用

趋势确认组合

MA + MACD：用移动平均线确认趋势方向，用MACD寻找入场时机

  • 价格在MA上方 + MACD金叉 = 强烈买入信号
  • 价格在MA下方 + MACD死叉 = 强烈卖出信号

超买超卖组合

RSI + 布林带：双重确认超买超卖状态

  • RSI超买 + 价格触及布林带上轨 = 卖出信号
  • RSI超卖 + 价格触及布林带下轨 = 买入信号

交易策略

趋势跟随策略

移动平均线交叉策略

使用两条不同周期的移动平均线，通过它们的交叉来产生买卖信号。

参数设置：
  • 快线：10日EMA
  • 慢线：20日EMA
  • 确认：成交量配合
交易规则：
  • 买入信号：快线上穿慢线(金叉)
  • 卖出信号：快线下穿慢线(死叉)
  • 止损设置：入场价的2-3%
  • 止盈设置：风险收益比1:2
优点：
  • 简单易懂
  • 适合趋势市场
  • 自动化程度高
缺点：
  • 震荡市场表现差
  • 存在滞后性
  • 假信号较多

突破策略

当价格突破重要的支撑或阻力位时进行交易。

阻力突破

价格向上突破重要阻力位

  • 确认：成交量放大
  • 入场：突破后回踩确认
  • 止损：突破点下方
支撑跌破

价格向下跌破重要支撑位

  • 确认：成交量放大
  • 入场：跌破后反弹确认
  • 止损：跌破点上方

震荡交易策略

区间交易策略

在明确的支撑阻力区间内进行高抛低吸操作。

识别条件：
  • 明确的水平支撑阻力
  • 多次测试未突破
  • 区间幅度足够大
  • 成交量相对稳定
交易规则：
  • 买入：接近支撑位时买入
  • 卖出：接近阻力位时卖出
  • 止损：突破区间边界
  • 止盈：区间另一端

风险管理

风险管理原则

2%规则

单笔交易风险不超过总资金的2%

示例：

总资金：$10,000

最大风险：$200

如果止损5%，最大仓位：$4,000

风险收益比

确保潜在收益至少是风险的2倍

示例：

入场价：$100

止损价：$95 (风险$5)

止盈价：$110 (收益$10)

风险收益比：1:2 ✓

分散投资

不要把所有资金投入单一品种

  • 不同币种分散
  • 不同策略分散
  • 不同时间分散
  • 保留现金储备

交易心理

控制情绪

  • 制定交易计划并严格执行
  • 不要因为恐惧或贪婪改变策略
  • 接受亏损是交易的一部分
  • 保持冷静和客观的判断

避免常见错误

  • 不要追涨杀跌
  • 不要频繁交易
  • 不要重仓单一品种
  • 不要忽视止损
